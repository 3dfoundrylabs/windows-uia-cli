# Privacy Policy

**windows-uia-cli** by 3D Foundry Labs Ltd.

Last updated: March 14, 2026

## Overview

windows-uia-cli is a local-only Windows UI automation tool. It runs entirely on your machine and is designed with privacy as a default.

## Data Collection

This tool **does not collect, transmit, or store any personal data**. Specifically:

- No analytics or telemetry
- No network requests of any kind
- No user accounts or authentication
- No cookies, tracking, or fingerprinting

## Local Data

The tool creates the following temporary files on your machine during normal operation:

- **Server PID file** in `%TEMP%` — tracks the background PowerShell process for auto-restart
- **Screenshots** in `%TEMP%` — only when you explicitly request a screenshot command

These files remain on your local machine and are not transmitted anywhere.

## Third-Party Services

This tool does not integrate with or send data to any third-party services.

Updates to this policy will be reflected in this file within the repository. The "Last updated" date at the top indicates the most recent revision.
